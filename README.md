# README

## 開発言語
* Ruby 3.0.1
* Rails 6.1.6 

## 就業Termの技術
* devise
* フォロー機能
* メッセージ機能

## カリキュラム外
* onesignal

## 実行手順
* \$ git clone git@github.com:moritamasak/time_finder.git
* \$ cd time_finder
* \$ bundle install
* \$ yarn install
* \$ rails db:create && rails db:migrate
* \$ rails s
